# GitHub Pages设置指南

## 步骤1：进入GitHub仓库设置
1. 打开浏览器，访问你的GitHub仓库页面：https://github.com/xingyuchen052-boop/meloway-station
2. 点击顶部导航栏中的 "Settings" 选项

## 步骤2：配置GitHub Pages
1. 在左侧导航栏中，点击 "Pages" 选项
2. 在 "Build and deployment" 部分：
   - Source：选择 "Deploy from a branch"
   - Branch：选择 "gh-pages"，并选择 "/ (root)"
3. 点击 "Save" 按钮

## 步骤3：等待部署完成
- GitHub Actions工作流会自动运行，构建并部署你的网站
- 部署完成后，你会在GitHub Pages设置页面看到网站的URL

## 步骤4：测试网站
- 打开生成的GitHub Pages URL，测试网站是否正常访问
- 网站URL格式通常为：https://username.github.io/repository-name
- 对于你的仓库，URL应该是：https://xingyuchen052-boop.github.io/meloway-station
